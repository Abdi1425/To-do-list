#  To-Do List Application

A simple and interactive To-Do List application built using **HTML**, **CSS**, and **JavaScript**. This project helps users manage tasks efficiently by allowing them to add, complete, search, and remove tasks dynamically.

---

#  Features

- Add new tasks
- Display tasks dynamically
- Mark tasks as completed
- Clear completed tasks
- Live search/filter functionality
- Responsive and clean user interface

---

#  What You Will Learn

In this project, you will explore the process of building a functional task management interface using HTML, CSS, and JavaScript.

You will understand how to:

- Add tasks dynamically
- Display and update a task list
- Toggle task completion
- Clear completed tasks
- Implement live task search
- Manipulate the DOM using JavaScript
- Handle user interactions with event listeners
- Work with arrays for task management

---

#  Learning Objectives

After completing this project, you will be able to:

## 1. Task Management Implementation
- Create a functional task management interface
- Add and display tasks dynamically
- Toggle task completion status
- Clear completed tasks

## 2. DOM Manipulation
- Dynamically create and modify HTML elements
- Update webpage content in real time
- Manipulate the Document Object Model (DOM)

## 3. Event Handling & User Interaction
- Implement event listeners
- Handle user actions such as:
  - Adding tasks
  - Completing tasks
  - Searching tasks

## 4. Front-End Development Principles
- Apply CSS styling and layout techniques
- Build responsive and user-friendly interfaces
- Integrate JavaScript functionality into web pages
